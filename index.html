VCFO Terminal — User Guide
Financial Health & Acquisition Screener · Data: UC AB via allabolag.se + Claude AI
🏢 Feature A — CFO Operational Audit
The CFO Audit acts as an internal financial controller to identify performance leaks. It evaluates three categories:

Revenue Quality — Flags rejected ROT/RUT tax deductions, unusual revenue concentrations, and income volatility versus prior periods.

Efficiency Drift — Calculates Return on Sales (Operating Profit ÷ Net Sales). Compares current ROS against the prior year to identify margin compression or improvement. A >15% ROS improvement or decline triggers a flag.

Aggressive Payouts — Flags as "Liquidity Concern" when dividends exceed annual net profit (Dividends > Net Profit), as this drains the equity buffer and may signal aggressive distribution ahead of distress.
🎯 Feature B — M&A Acquisition Fit Screener
The M&A Screener evaluates a company for investment readiness across four dimensions:

Compliance Due Diligence — Scans the auditor's report for "Anmärkning" (Remark) flagging late annual report filings per ABL 7 kap. 10§. This creates a legal compliance risk for acquirers.

Asset Lifecycle — Checks whether fixed assets (Inventarier, Maskiner) are fully depreciated (Book Value = 0 or residual <5%), indicating a new owner faces immediate and material CapEx requirements.

Debt Maturity — Evaluates long-term liabilities (Långfristiga skulder), specifically those due in >5 years, to assess whether post-acquisition cash flows are constrained.

Scoring Model — The Buy/Pass score (0–100) aggregates: Profitability (25 pts), Compliance (25 pts), Balance Sheet strength (25 pts), Efficiency (25 pts). >65 = BUY, 40–65 = HOLD, <40 = PASS.
🔌 API Configuration
allabolag.se (UC AB) — The primary data source. Financial data is sourced from UC AB's processed annual reports, accessed via allabolag.se. Organisation numbers follow the format 556000-1234.

Anthropic Claude API — Powers the AI CFO Audit narrative and M&A recommendation. Set your API key in ⚙ API Keys. The tool uses claude-sonnet-4-20250514 for structured JSON output.

CORS Note — The Bolagsverket API may require a proxy in some environments. The tool attempts a direct fetch first and falls back to demo data if blocked.

📡 Live Sector Benchmarks
Pulls real median ratios from Statistics Sweden (SCB) · Free data, no login needed · Updates the Compare tab automatically
What this tab does
Connects to SCB's open statistics API and downloads the official sector medians for whichever industries you select. Once fetched, the Compare tab automatically uses these real numbers instead of built-in estimates.
When to refresh
SCB publishes updated figures once a year (typically November). Just come back here, click Fetch from SCB, and everything updates. The app remembers the last fetch so it works offline too.
Data source
SCB Branschnyckeltal (NV0109) — 46 ratios across ~800 Swedish industry branches. Free under CC0 licence. Covers all Swedish limited companies filing annual reports.
